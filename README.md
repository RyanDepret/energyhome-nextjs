# EnergyHome AI — Next.js

Plateforme web d'analyse énergétique immobilière (DPE) construite avec Next.js. Intègre un simulateur de classe énergétique en lien avec la loi Climat et Résilience (interdiction de location des biens classés G depuis 2025, F en 2028).

## 🌐 Démo en ligne

[https://energyhome-nextjs.vercel.app](https://energyhome-nextjs.vercel.app)

## Technologies utilisées

- Next.js 15
- React 19
- CSS Modules
- Font Awesome
- Vercel (déploiement)
- next-intl

## Fonctionnalités

- Landing page responsive avec hero immersif
- Simulateur DPE interactif
- Navigation multi-pages avec App Router
- SEO optimisé avec metadata Next.js
- Menu burger responsive
- Design glassmorphisme moderne
- Traduction FR/EN
